# 竞争情报研究技能 / Competitive Intelligence Research Skill

## 研究重点 / Research Focus

### 产品与服务分析 / Product & Service Analysis
- 功能对比与独特卖点
- 技术栈与创新能力
- 产品路线图指标和近期发布

### 市场地位 / Market Position
- 市场份额估计和趋势
- 地理覆盖和扩展模式
- 目标客户细分

### 财务指标 / Financial Indicators
- 收入和增长指标（如公开）
- 融资历史和投资者概况（如适用）
- 定价策略和商业模式

### 战略洞察 / Strategic Insights
- 近期新闻稿和公告
- 合作伙伴和收购活动
- 高管领导层变动

## 数据收集优先级 / Data Collection Priority

1. **市场数据 / Market Data**: 市场规模、份额、增长率
2. **财务数据 / Financial Data**: 收入、利润、估值、融资
3. **技术指标 / Technical Metrics**: 性能参数、技术规格对比
4. **竞争格局 / Competitive Landscape**: 关键玩家排名、市场地位
5. **趋势预测 / Trend Forecasts**: 增长预测、技术趋势

## 搜索策略 / Search Strategy

- 执行 5-10 个目标性搜索查询
- 优先官方来源（公司网站、投资者关系）
- 补充行业报告和分析师报道
- 收集客户评论和满意度数据
- 捕获社交媒体存在和情感

## 输出规范 / Output Specification

- 章节标题：中文 / English
- 关键术语：提供双语版本
- 数据表格：英文列标题 + 中文行标签

## 质量标准 / Quality Standards
- 每个研究文件至少包含 10-15 个具体统计数据

- 使用表格呈现比较数据
- 避免模糊陈述，使用精确数字
