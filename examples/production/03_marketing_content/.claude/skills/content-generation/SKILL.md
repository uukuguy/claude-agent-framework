# Content Generation Skill

## Objectives

Help create high-quality marketing content that:
- Captures audience attention quickly
- Communicates value propositions clearly
- Drives desired user actions
- Optimizes for search visibility

## Writing Frameworks

### AIDA Framework
1. **Attention**: Hook with compelling headline or opening
2. **Interest**: Build curiosity with relevant benefits
3. **Desire**: Create emotional connection with outcomes
4. **Action**: Clear, compelling call-to-action

### PAS Framework
1. **Problem**: Identify pain point
2. **Agitate**: Amplify the problem
3. **Solution**: Present your solution

### FAB Framework
1. **Features**: What it does
2. **Advantages**: Why it matters
3. **Benefits**: What user gains

## Headline Formulas

### Number Headlines
- "7 Ways to [Achieve Desired Outcome]"
- "[Number] [Adjective] [Topic] Tips That [Promise]"

### How-To Headlines
- "How to [Do Something] Without [Pain Point]"
- "How [Target Audience] Can [Achieve Goal]"

### Question Headlines
- "Are You Making These [Topic] Mistakes?"
- "What If You Could [Desired Outcome]?"

## SEO Optimization

### Keyword Integration
- Primary keyword in headline/title
- Secondary keywords in subheadings
- Natural density (1-2% for primary)
- Long-tail keywords in body text

### Structure for SEO
- Clear H1, H2, H3 hierarchy
- Short paragraphs (2-3 sentences)
- Bullet points for scannability
- Meta-friendly first paragraph

## CTA Patterns

### Action-Oriented
- "Start Your Free Trial"
- "Get Your Guide Now"
- "Join [Number]+ [Users]"

### Benefit-Focused
- "Boost Your [Metric] Today"
- "Unlock [Benefit]"

### Urgency-Based
- "Limited Time Offer"
- "Only [Number] Spots Left"

## Content Length Guidelines

| Content Type | Recommended Length |
|-------------|-------------------|
| Blog Post | 1,500-2,500 words |
| Landing Page | 500-1,000 words |
| Email | 100-300 words |
| Social Post | 50-150 characters |
| Product Description | 100-300 words |

## Quality Checklist

- [ ] Clear value proposition in first 100 words
- [ ] Compelling headline following proven formulas
- [ ] Keywords integrated naturally
- [ ] Strong, specific CTA
- [ ] Scannable structure
- [ ] Appropriate tone for audience
