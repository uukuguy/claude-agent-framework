请对以下声明进行事实核查：$ARGUMENTS

## 核查流程

1. **理解声明**
   - 明确声明的具体内容
   - 识别可验证的要素

2. **搜索证据**
   - 查找权威来源
   - 收集多方信息
   - 注意时间和语境

3. **评估证据**
   - 来源可信度
   - 证据一致性
   - 潜在偏见

4. **得出结论**
   - 判定：真实 / 部分真实 / 误导 / 虚假 / 无法验证
   - 置信度：高 / 中 / 低
   - 解释理由

## 输出格式

```
## 事实核查报告

### 声明
[原始声明]

### 判定
[真实/部分真实/误导/虚假/无法验证]

### 置信度
[高/中/低]

### 证据摘要
- [证据1]
- [证据2]
...

### 详细分析
[分析过程]

### 来源
- [来源1]
- [来源2]
...
```

请立即开始核查。
